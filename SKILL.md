# 🌱 SKILL.md  
## AI-Driven Crop Disease Prediction & Management System

---

# 🎯 Project Vision

This is not a basic CRUD web application.

This is a **full-stack AI-powered agricultural intelligence platform** designed to:

- Detect crop diseases using CNN (MobileNetV2)
- Assess severity levels
- Provide AI-based treatment recommendations
- Integrate weather-based disease risk prediction
- Support multilingual users
- Enable admin-controlled disease knowledge management
- Deliver a distinctive, production-grade frontend interface

This system is intentionally engineered — not generated.

---

# ⚛ Frontend Engineering (React + Vite)

## 🎨 Official UI Theme  
### Agri-Tech Intelligence Console

The frontend follows a **dark industrial-agricultural aesthetic** blending organic crop tones with analytical precision.

This system must feel like:

> A precision agricultural intelligence console — not a generic SaaS dashboard.

---

## 🎨 Color System

Primary Background:

#0F1A14 (Deep forest black-green)


Accent Green:

#2F8F5B (Organic crop green)


Warning Amber:

#E4B363 (Dry soil tone)


Severe Red:

#B33A3A (Disease severity indicator)


Text:

#EAEFEA


No purple gradients.  
No generic AI color clichés.

---

## 🧵 Typography Strategy

Display Font:
- Cormorant Garamond (Editorial agricultural tone)

Body Font:
- IBM Plex Sans (Industrial clarity)

Avoid:
- Inter
- Roboto
- Arial
- Generic system fonts

## 🧱 Spatial & Layout Design

- Vertical sidebar navigation
- Data-card based dashboard
- Large editorial headers

## 🎥 Motion Design Philosophy

- Staggered fade-in page load
